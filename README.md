# AI 인터렉션 기반 교내 프로젝트 지식 아카이브

## 프로젝트 개요

충북대학교 내 다양한 정보기술 프로젝트 산출물이 GitHub 조직 저장소, 개인 저장소, 개인 폴더 등에 분산되어 축적·공유되지 못하는 문제를 해결하기 위해, **프로젝트 자료를 통합 저장하고 자연어 기반으로 탐색·추천할 수 있는 지식 아카이브 플랫폼**을 구축하는 프로젝트입니다.

이 플랫폼은 단순한 보관소를 넘어, 사용자가 자연어로 질문하면 관련 프로젝트, 기술 스택, 유사 사례, 난이도, 관련 데이터셋 등을 찾아주고 비교·추천까지 제공하는 **대화형 AI 아카이브**를 목표로 합니다.

---

## 프로젝트 필요성

### 1. 기존 문제점

* 학기마다 생성되는 교내 프로젝트 결과물이 체계적으로 보존되지 않음
* 과거 팀프로젝트, 캡스톤, 과제 결과물에 대한 접근성이 낮음
* 졸업생 우수 프로젝트가 시간이 지나며 사라지거나 개인 저장소에만 남는 경우가 많음
* 선·후배 간 개발 지식 전승과 학습 연속성이 약화됨

### 2. 기존 솔루션의 한계

현재 존재하는 대학 프로젝트 아카이브는 대체로 다음 두 유형입니다.

* GitHub 기반 저장소형
* 웹 기반 전시·목록형 플랫폼

하지만 두 방식 모두 다음 한계를 가집니다.

* 검색 의도를 이해하는 기능 부족
* 자연어 기반 질의응답 및 추천 기능 부재
* 기술 스택, 데이터셋, 유사 사례를 연결하는 지능형 탐색 기능 부족
* 학습자 중심 인터랙션이 부족함

---

## 프로젝트 목표

### 1. 프로젝트 아카이브 플랫폼 구축

교내 정보기술 프로젝트를 한곳에 통합 저장하고, 학습·탐색·지식 전승을 지원하는 플랫폼을 구축합니다.

### 2. 자연어 기반 탐색·추천 시스템 개발

사용자가 자연어로 질문하면 프로젝트 도메인, 기술 스택, 난이도, 유사 사례를 기반으로 적절한 프로젝트를 탐색·추천할 수 있는 지능형 시스템을 개발합니다.

### 3. 스프링 기반 서버 및 데이터 관리 구조 설계

프로젝트 정보 수집, 저장, 조회, 검색, 추천 기능을 안정적으로 운영할 수 있도록 백엔드 서버와 데이터 흐름을 체계적으로 설계합니다.

### 4. 유지보수성과 확장성 확보

후속 개발자와 후배들이 쉽게 이해하고 확장할 수 있도록 문서화와 모듈화를 고려한 구조를 설계합니다.

### 프로젝트 아카이브

* 프로젝트 정보 구조화 저장
* 프로젝트 문서, 산출물, 코드 보관
* 프로젝트 목록 및 상세 조회
* 연도, 과목, 기술 스택 등 필터링 기능

### AI 기반 검색 및 추천

* 자연어 질의 기반 프로젝트 탐색
* 키워드 검색 + 벡터 검색 결합
* 유사 프로젝트 추천
* 프로젝트 요약 및 비교 제공
* 후속 질문의 맥락을 유지하는 대화형 검색 흐름

* README, 문서, 코드 파일 분석
* 도메인, 기술 스택, 난이도, 주요 기능 자동 추출
* 검색 및 추천 정확도 향상을 위한 구조화 처리

### 학습 지원 기능

* 관련 프로젝트 추천
* 기술 스택 기반 참고 자료 연결
* 주요 코드 구조 및 프로젝트 설명 자동 요약
* 학습 목적 기반 가이드 제공

### 플랫폼 활성화 기능

* 프로젝트 업로드 유도 구조 설계
* 배지, 포인트 등 게이미피케이션 요소 도입 가능
* 학과/팀 단위 참여 활성화 구조 확장 가능

## 시스템 구성

### 저장 구조

* **객체 스토리지**: MinIO
* **관계형 데이터베이스**: PostgreSQL
* **검색 엔진**: Elasticsearch 또는 OpenSearch
* **벡터 검색**: pgvector 또는 Qdrant

### AI/검색 파이프라인

* 임베딩 모델: `bge-small`, `e5-small`
* 자연어 질의 해석: LLM 기반 의도 분석
* 검색 방식:

  * 메타데이터 검색
  * 키워드 검색
  * 벡터 유사도 검색
* 응답 방식: RAG 기반 요약 및 추천

### 백엔드 / 프론트엔드

* **Backend**: Spring Boot
* **Frontend**: React
* **API 문서화**: Swagger 등 활용 예정

## 기대 효과

### 1. 서비스 아키텍처 이해 향상

AI, 검색 엔진, DB, 백엔드, 프론트엔드가 연동되는 전체 서비스 구조를 직접 경험하며 시스템 설계 이해도를 높입니다.

### 2. 자연어처리 및 머신러닝 역량 강화
